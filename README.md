## Описание программы
Это GUI-приложение «GitHub User Finder» с использованием GitHub API, сохранением избранных пользователей и Git.
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
## Как запустить
```bash
git clone https://github.com/user73312282/github-user-finder.git
cd GitHub-User-Finder
python main.py
